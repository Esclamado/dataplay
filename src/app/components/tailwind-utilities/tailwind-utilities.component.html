Tailwind Utilities
Breakpoints
breakpoint:class (e.g. sm:text-white md:text-black)
Grids
Grid column
Defines columns for grid layout.
parent div class: grid (display:grid)   grid-cols-3 (3 columns)    gap-4(to specifiy gaps)
child element class: col-span-3
child element class: col-span-2
child element class: col-span-1
child element class: col-span-1
child element class: col-span-2
Grid Column starting and ending lines
Sets a grid item location within the grid column.
parent div class: grid (display:grid)   grid-cols-6 (6 columns)    gap-4(to specifiy gaps)
child element class: col-start-2 col-span-4
child element class: col-start-1 col-end-3
child element class: col-end-7 col-span-2
child element class: col-start-1 col-end-7
Grid rows
Defines rows for grid layout.
parent div class:
grid (display:grid)   grid-rows-3 (3 rows)    grid-flow-col(where to put next div when row is full)    gap-4(to specifiy gaps)
child element class: row-span-3
child element class: row-span-1 col-span-2
child element class: row-span-2 col-span-2
Grid Row starting and ending lines
Sets a grid item location within the grid row.
parent div class: grid (display:grid)   grid-rows-3 (3 rows)    gap-4(to specifiy gaps)
child element class: row-start-2 row-span-2
child element class: row-end-3 row-span-2
child element class: row-start-1 row-end-4
Flex
Sets element to be a flex container.
Flex Width
Sets the width of flex container.
parent div class:    flex (display:flex) Width automatically set to 100%
Flex Container
parent div class:    inline-flex (display:inline-flex) Width depends on the width of it's content
